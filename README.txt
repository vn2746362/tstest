== Chạy Project
1) Mở project với Android Studio 3.4.2~
2) Yêu cầu có hoặc cài đặt:
	- compileSdkVersion 28
	- gradle-5.1.1
3) Sync thư viện
4) Run Project với Emulator

== Chạy Unit Test
1) Vào pakage com.horizon.tstest.validators
2) Chọn PhoneValidatorTest, right click rồi Run 'PhoneValidatorTest'
